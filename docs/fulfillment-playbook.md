# Fulfillment Playbook

## Inputs

Collect these before starting:

- Company URL
- Work email
- Target customer
- Primary location or market, if relevant
- 2-4 competitors
- Any known positioning or service focus

## Audit Steps

1. Read the homepage, primary service/product pages, about page, pricing page if public, testimonials/case studies, and FAQ.
2. Write 10 buyer-intent prompts:
   - 3 category discovery prompts
   - 2 comparison prompts
   - 2 local or niche-fit prompts
   - 2 objection/risk prompts
   - 1 direct brand-understanding prompt
3. Test prompts in available AI/search-answer tools manually.
4. Record:
   - Whether the company appears
   - Which competitors appear
   - Whether the description is accurate
   - Which sources or public pages are referenced
   - What proof points are missing
5. Inspect the customer's public pages for:
   - Clear category language
   - Named services and buyer types
   - Locations served
   - Comparison language
   - Testimonials, outcomes, case studies, and credentials
   - FAQ coverage
   - Organization, LocalBusiness, Product, Service, Review, and FAQ schema where appropriate
6. Produce the report.

## Report Outline

Use a concise report with these sections:

- Executive snapshot
- Prompt test matrix
- Competitor visibility notes
- Accuracy issues
- Citation and proof gaps
- One-week action plan
- Retest prompt set

## Quality Bar

Every recommendation should be:

- Tied to a prompt result or public-page observation
- Specific enough for a website owner or marketer to implement
- Framed as improving discoverability and clarity, not guaranteeing rankings
